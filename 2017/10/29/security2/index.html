单变量线性同余

定理：令 a, b, n 为整数，且a>0, gcd(a, n)=d (1)若 d不能整除b, 则ax=b mod n 无解 (2)若 d能整除b, 记做d|b, 则ax=b mod n 有d个解

ax = b mod n的求解过程： >1.利用欧几里德算法求出d=(a, n)，若d 不能整除b, 则无解。 2.若d能整除b, 则令：a'=a/d,b'=b/d,n'=n/d; 因为 (a', n')=1, a'x' =b' mod n' 有唯一解，假设此解为x0 利用欧几里德算法求出a'模n'的乘法逆元(a')-1, x’= (a')-1 b' mod n' x0 = x' mod n 令x= x0 +(n/d)t mod n, t=1,2, …, d-1, 即可求出剩下的d-1个解
